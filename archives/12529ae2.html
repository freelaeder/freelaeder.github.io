React + TypeScript - 类组件

01. 创建应用

目标：掌握如何使用 create-react-app 创建支持 TypeScript 的 React 应用的方式

使用 React 官方提供的 create-react-app 脚手架工具即可创建支持 TypeScript 的 React 应用。

我们只需要在命令中添加 —template typescript 选项即可创建 TypeScript 项目。

在 TypeScript 项目中包含 JSX 代码的文件后缀名称为 tsx，其他不包含 JSX 代码的文件后缀名称为 ts。

02. Component 泛型类

目标：掌握如何通过 Component 泛型类标注组件 Props 对象和 State 对象的类型

Component 是一个泛型类，通过它的第一个泛型参数可以标注组件 Props 对象的类型，第二个参数可以标注组件 State 对象的类型。

03. 标注 React 元素的类型

(1) 什么是 React 元素

通过 React.createElement 方法创建的元素、通过 JSX 语法创建的元素都被叫做 React 元素。(虚拟 DOM 对象)

(2) 如何标注 React 元素的类型

使用 ReactElement 和 JSX.Element 都可以标注 React 元素的类型。

ReactElement 主要用于描述虚拟 DOM 对象的类型。

JSX.Element 主要用于描述通过 JSX 语法创建的元素的类型。

JSX 被编译之之后其实就是虚拟 DOM 对象，而且在编译之后 JSX 类型被自动映射为 ReactElement 类型，所以两个类型几乎通用。

04. ReactPortal (了解)

ReactPortal 表示 ReactDOM.createPortal 方法的返回值的类型。

05. ReactNode

目标：掌握 React.ReactNode 类型的使用方式

ReactNode 表示一个可以渲染的任意值，包括布尔值、数字、字符串、元素和数组等，它是 React 中最泛化的类型。

06. ReactFragment

ReactFragment 表示一组可以被遍历渲染的任意值，它只被用于标注组件 children 属性的类型。

07. CSSProperties

目标：掌握 CSSProperties 类型的使用方式

CSSProperties 类型用于标注 styles 属性值的类型，它对样式对象中的属性及属性值的类型进行约束。

08. ChangeEvent

目标：掌握 ChangeEvent 类型的使用方式

ChangeEvent 用于标注表单项 change 事件的事件对象的类型。

09. MouseEvent

目标：掌握 React.MouseEvent 类型的使用方式

MouseEvent 用于标注鼠标事件对应的事件对象的类型。

10. KeyboardEvent

KeyboardEvent 用于标注键盘事件对应的事件对象的类型。

11. FormEvent

FormEvent 用于标注表单提交事件对应的事件对象的类型。

12. UIEvent

UIEvent 是一个通用的和用户界面相关的事件的事件对象的类型，MouseEvent、KeyboardEvent 等都是它的子类。

开发者在标注事件对象的类型时有子类优先使用子类。

一般我们使用 UIEvent 标注滚动事件的事件对象类型、加载事件的事件对象类型等。
